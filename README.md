# Dia C Code exporter
State Machine Exporter for [Dia Diagrams](http://dia-installer.de/download/index.html).

# Instruction
Place `fsm_c_code_export.py`  with the work of [Unai EstÃ©banez Sevilla](http://www.sourcepole.ch/assets/2012/6/5/uml_stm_export.py) in your home directory `~/.dia/python/`.

Now you can use Dia `File -> Export -> Select Filetype (dropdown) -> State Machine C-Code and Headerfile -> Save` to export your FSM.
